welcome to the chroma zone

This is where I collect various apps, utilities, and other projects I've worked on. Everything listed here is open source and free for you to try.

Projects Overview

  • ChromaFiler
    An experimental file manager for Windows, borrowing ideas from Spatial file managers and Miller Column browsers.
  • N-Space
    A mobile app for building 3D interactive worlds. It has a unique interface built for rapid design and modification, and it has a logic system for making games. WIP successor: N-Space 2
  • Aodix Enhanced
    A fork of the Aodix music sequencer to fix longstanding bugs and continue development.
  • WingEd
    A 3D modeling tool for low-poly level design.
  • AR Recorder
    Use your phone as a 6DOF controller for Blender.
  • SketchUp Extensions
    A collection of extensions for SketchUp 2017+. The most useful of these is Hide Back Faces, which implements back-face culling.
  • Tapedeck
    An audio editor designed to be controlled with a MIDI DJ board and no screen.
